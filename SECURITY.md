If you discover a security vulnerability related to:

- the AntartideCoin website (antartidecoin.com),
- the AntartideCoin wallet,
- the AntartideCoin node, explorer or mining pool,

**please do NOT open a public GitHub issue.**

Instead, send a detailed report to:

- `info@antartidecoin.com`

If possible, include:

- a description of the issue,
- how to reproduce it,
- the environment (OS, wallet version, etc.),
- any logs or screenshots that can help.

We will review your report as soon as possible and contact you back.
Please do not publicly disclose the vulnerability until it has been fixed.
